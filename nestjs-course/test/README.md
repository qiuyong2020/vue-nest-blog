# 创建 Nest.js 项目

## Description

- 命令使用
- 模块`module`、控制器`controller`、路由`router`、服务`service`
- 路由简单访问
- 依赖注入

## Installation

```bash
$ npm install @nestjs/cli
$ nest new [project-name]
>>> pnpm
$ pnpm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
