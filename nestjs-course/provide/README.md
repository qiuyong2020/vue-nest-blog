# `Service/provider` 服务提供者

## Description
- 服务提供者注册
- 类的注册方式
- 基本类型的提供者注册
- 动态注册服务提供者
- 通过服务动态加载配置项
- 使用工厂函数注册
- 模块共享服务
- 异步服务提供者

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
